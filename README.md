# EV Charging Station Analytics using Power BI

## Overview

This project analyzes Electric Vehicle (EV) charging station data using **Power BI** to gain insights into charging behavior, station utilization, energy consumption, charging costs, and operational performance. Interactive dashboards built with **Power Query** and **DAX** help identify usage trends, monitor KPIs, and support data-driven decision-making for efficient EV infrastructure planning.

## Objectives

- Analyze charging station utilization and energy consumption.
- Identify peak charging hours and user behavior.
- Compare charger type usage and charging costs.
- Evaluate the impact of vehicle age and environmental factors on charging.
- Forecast future charging demand.

## Tools Used

- Power BI Desktop
- Power Query
- DAX
- Microsoft Excel

## Dashboard Features

- Interactive KPI Dashboard
- Charging Station Utilization Analysis
- Energy Consumption Trends
- User Behavior Analysis
- Charger Type Comparison
- Charging Cost Analysis
- Demand Forecasting

## Key Insights

- Identified peak charging periods and station utilization patterns.
- Compared charging behavior across different user types.
- Analyzed the impact of vehicle age and temperature on charging efficiency.
- Evaluated energy consumption and charging costs to support operational planning.
